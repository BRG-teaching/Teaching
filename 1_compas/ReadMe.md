Focus:

- Teaching programming, with a focus on object-oriented programming, is a core aspect: geometric topology, hierarchy, and
non-hierarchy concepts in parametric, modeling are explored to develop, robust approaches. It is designed as an “introduction”
into the program, and provides an intensive skills-building bootcamp format featuring full-time lectures and tutorials supported by a team of researchers. The curriculum is strategically designed to prioritize the rapid acquisition of digital skills.
- geometry processing
- algorithms
- optimisation strategies
- discretisation
- data structures
- developing, structuring and documenting proper code and robust “software” tools.

Outcomes:
- develop skills in coding using COMPAS Python Framework
- learn to apply digital tools and processes for digital design and fabrication
- be able to work in teams of 3 to 4 on computational design projects
- define your personal learning journey and skills development focus
